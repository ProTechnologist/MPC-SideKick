# MPC-SideKick
A smart, auto-hiding side panel for MPC-BE that provides library navigation, playlist management, and quick media access.

## Features
- **Auto-Hiding Side Panels**: Library and Playlist panels slide in from the edges when needed.
- **Persistent Playlist**: Queue your videos and manage them easily with drag-and-drop.
- **Fuzzy Search**: Quickly find videos in your library or playlist.
- **Deep Carbon UI**: Modern, clean design with professional aesthetics.
- **MPC-BE Integration**: Seamlessly anchors to your player window even in fullscreen mode.

## How to run
1. Ensure MPC-BE is installed at `C:\Program Files\MPC-BE\mpc-be64.exe`.
2. Run `dotnet run` to start the application.
3. Use the Library (left edge) to manage your media.
4. Use the Playlist (right edge) for your active queue.
